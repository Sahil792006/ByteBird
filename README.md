# 🐦 ByteBird

ByteBird is a **micro-blogging and blogging platform** inspired by Twitter and Medium.
Users can share **short posts (Bytes)** or **long-form blogs (Birds)**, interact with others, and discover trending ideas.

The goal of this project is to build a **modern blogging platform from scratch** while learning full-stack development.

---

## 🚀 Features

* 🔐 User Authentication (Signup / Login)
* 📝 Create Posts (Short Bytes & Long Blogs)
* ❤️ Like Posts
* 💬 Comment on Posts
* 👥 Follow / Unfollow Users
* 🔎 Explore Trending Blogs
* 📑 Bookmark Posts
* 📊 Blog Analytics (views & engagement)
* 🌙 Dark Mode (planned)

---

## 🧠 Concept

ByteBird combines two types of content:

**Bytes 🐦**

* Short thoughts or quick updates

**Birds 🪶**

* Full blogs or long-form posts

This allows users to share both **quick ideas and detailed articles** in one place.

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/bytebird.git
```

Navigate into the project

```
cd bytebird
```

Install dependencies

```
npm install
```

Run the server

```
npm start
```
